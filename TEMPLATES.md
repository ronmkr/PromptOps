# Template Changelog

This document tracks major updates to the expert prompt templates in the PromptBook library.

## [1.0.0] - 2026-03-26
### Initial Public Release
- 160+ expert prompts across Engineering, Security, AI, DevOps, and more.
- Standardized metadata schema (description, version, tags, etc.).
- Support for dynamic context (`{{args}}`, `{{file}}`, `{{language}}`).
- Conditional extraction using `<if language="...">` blocks.
- PII masking support for GDPR compliance.
