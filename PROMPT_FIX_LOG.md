# Prompt Library Fix Log

Track of systematic cleanup, consolidation, and metadata standardization of the PromptOps library.

## Standards Applied
- **Description**: Under 150 characters, complete (no `...`), and clear.
- **Tags**: Exactly ONE tag matching the parent directory name.
- **Consolidation**: Redundant/overlapping prompts merged into "Master" or "Specialist" guides.
- **Categorization**: Prompts moved to the most appropriate functional directory.
- **Naming**: Consistent use of "Agent" for personalities and "Specialist" for technical guides.
